# three-tier-project

#### React + Node.js + MongoDB - Job Board Real-time App (built from scratch)

→ Client: React + Vite + TypeScript + Tailwind

→ Server: Node.js + Express + MongoDB + WebSocket

→ Dockerized for linux/amd64

→ Pushed to private ECR registry on AWS

→ Running on Kubernetes (EKS) in us-east-1

→ 2 worker nodes (t3.small)

→ ArgoCD watching GitHub for auto-deployments

→ AWS ALB routing public internet traffic

→ Live URL accessible from anywhere in the world
